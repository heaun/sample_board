Console Program

January 24, 2017 by Eun

Hello World - Console Program

using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
        }
    }
}

  • c#은 대소문자를 구분한다. e.g. using System, Main(), Console.WriteLine ...
  • namespace : 선언 및 시작점. 프로그램을 전체적으로 구조화하고 그 구조의 영역을 나타내는 부분. 이 구조를 다른 프로그램에서 사용하기 위해서 using 이라는 키워드를 사용하는 것이다.
  • class : 프로젝트는 항상 반드시 하나의 클래스를 가지고 있어야 한다. 데이터 타입 구조체로서 하나의 데이터 구조체를 생성하는 부분. 멤버 변수, 멤버 메소드(함수) 등을 둘 수 있고 다른 클래서의 멤버 변수나 멤버 메소드를 사용하기 위해선 객체, 즉 인스턴스를 생성하여 접근 및 사용하는데 그 키워드는 new 이다.
  • Main() : 프로젝트는 항상 반드시 하나의 메소드를 가지고 있어야 한다. 하나의 데이터 타입 구조체 내부에서 실제 비지니스 로직 및 어떤 행위 등을 담당하는 부분으로 개발자가 실제로 코딩해야 할 부분.
  • 콘솔 프로그램 시작 : Ctrl + F5
  • 콘솔 프로그램 종료 : F9
